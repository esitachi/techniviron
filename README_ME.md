# About Me / Project Summary

Hi — I'm Eshaan. This repository contains a realtime AI backend that demonstrates
WebSocket sessions, LLM streaming, and persistence with Supabase.

If you want to run or extend this project locally:

- Create a `.env` with `OPENAI_API_KEY`, `SUPABASE_URL`, and `SUPABASE_SERVICE_ROLE_KEY`.
- Install dependencies: `pip install -r requirements.txt`.
- Run the server: `uvicorn app.tech_main:app --reload --port 8001`.

Feel free to reach out or open issues if you want help improving features,
adding tests, or wiring CI/CD.
